# TO DO APPLICATION

This app allows users to track the dos.

#### Instructions
1.Download the source code
```bash
git clone https://github.com/xeroctation/to-do-app.git
```

or just download the *.zip file

2.Install description
```bash
npm install
```

3.Run the app
```bash
node app
```

#### Deps list
- express
- pug
